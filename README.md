# Health is Responsibility (صحّتك مسؤولية) 🩺

A simple, bilingual educational platform built to raise health awareness and share basic first-aid steps within the community. 

---

## 🌟 Why I Built This Project
My name is Isra, I am 14 years old, and my biggest dream is to become a neurosurgeon in the future. During my daily life, I noticed that many people around me panic or don't know how to act when faced with a simple injury or a sudden health situation, mostly due to a lack of basic health awareness. 

I decided to use my time during the school holidays to do something helpful. I built this website to provide simple, reliable, and quick information that anyone can read. My goal is to help spread medical awareness, hoping that one day, this knowledge might help someone save a life or help a person in need before medical professionals arrive.

---

## 🚀 Website Features

*   **Simple Symptom Guide:** A helpful box where you can type what you feel (like a headache or fever) or click on ready tags to get an immediate educational note about the situation.
*   **First Aid Steps:** Clear, easy-to-follow instructions for handling common situations such as severe bleeding, burns, choking, fainting, fractures, and basic CPR.
*   **Health Literacy Articles:** Informative cards that explain common health topics, like understanding blood pressure numbers, diabetes warning signs, infection prevention, and recognizing stroke symptoms (using the FAST method).
*   **Bilingual Interface:** Supports both Arabic and English seamlessly, so more people can access and benefit from the information.

---

## 🛠️ How I Developed It

*   **Code:** Written entirely by myself using **Visual Studio Code (VS Code)**.
*   **Technologies:** Built using classic **HTML5** for structure, **CSS3** for a clean layout and design, and **JavaScript** to handle the page switching and symptom guide logic.
*   **Sources:** All information and first-aid steps on this website are gathered from trusted global sources, specifically the **World Health Organization (WHO)** and **Mayo Clinic**.
*   **Hosting:** Hosted and deployed for free using **GitHub Pages**.
